BAB 1 — Executive Summary & Topologi Infrastruktur

Aplikasi beroperasi sebagai Single Page Application akademik universitas dengan model hybrid: frontend monolitik JavaScript untuk interaksi cepat berbasis localStorage, serta backend FastAPI + DuckDB untuk event monitor, pipeline, RAG, audit, export, backup, dan final validation.

function resolveLogin(role, username, password) {
  const u = normalized(username);
  const p = String(password ?? '').trim();
  const base = accounts[role];
  if (role === 'Mahasiswa') {
    if (base && base.username === username && base.password === password) return base;
    const student = state.students.find((s) => normalized(s.nim) === u && String(s.nim) === p);
    if (student) return accountFromStudent(student);
  }
  return null;
}

def get_current_user(credentials: HTTPAuthorizationCredentials = Depends(security)):
    token = credentials.credentials
    decoded = base64.b64decode(token).decode("utf-8")
    role, username = decoded.split(":", 1)
    return {"role": role, "username": username}

def actor_role(payload_role: str | None, user: dict[str, Any] | None) -> str:
    token_role = str((user or {}).get("role", "")).strip()
    if token_role:
        return token_role
    return str(payload_role or "System").strip() or "System"

BAB 3 — Integritas Struktural Front-End & Zero UI Mutation

Reuse komponen visual (`.card`, `.data-table`, `.status`, `.activity-list`, `.quick-actions`) menjaga konsistensi lintas role. Optimasi dilakukan pada engine render/event loop, bukan merusak kontrak UI utama.

BAB 4 — Rekayasa Anti-Jank & Performa Ekstrem

  • requestAnimationFrame untuk animasi sinkron frame.
  • IntersectionObserver untuk reveal on viewport.
  • Debounce command center dan batching localStorage writes.
  • Particle engine spatial grid untuk efisiensi.
